# Assignment-5-Part-A-Inheritance----PersonData-StudentData

This program addresses Chapters 2, 3, and 6.

Person Data and Student Data

Design a class named PersonData in PersonData.h and PersonData.cpp with the following member variables:

lastName
firstName
address
city
state
zip
phone
Write a constructor which populates all of the member variables and appropriate accessor and mutator functions FOR EACH OF THESE member variables.  The .h and .cpp files should be safeguarded with preprocessor directives.

Next, design a class named StudentData in StudentData.h and StudentData.cpp, which is derived from the PersonData class. The StudentData class should have the following member variables:

studentID
incomingFreshman

The studentID variable will be used to hold a unique integer for each student.  The incomingFreshman variable should be a boolean. It will be set to true if the student is an incoming freshman or false if the student is not an incoming freshman. Write a constructor which populates all of the member variables and appropriate accessor and mutator functions FOR EACH OF THESE member variables.

Finally, demonstrate an object of the StudentData class in a simple program and show that all of the member functions work.  Your test program should display to the screen what you are intending to test and should demonstrate that the test does in fact come out correctly.  If ALL your member functions are not tested, then you will not receive credit for them.

Your main program should demonstrate all the functionality of your program – every member function – without taking any input from the user.  You should have cout statements which tell me what is being tested and proves to me that what is being tested in fact works.  I should not have to type anything into the program except for clicking the run button.
